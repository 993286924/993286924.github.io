提权基础命令

ifconfig -a：Linux下查看IP地址

ipconfig /all：Windows下查看IP地址信息

netstat -ano：获取主机的端口开放情况，采用-anb时需要管理员权限

net start：查看系``统开放的服务

net stop servicename：停止服务

net start servicename：开启服务

tasklist /svc：获取进程名称、pid、服务

msinfo32：获取系统详细信息，如下将结果输出到文件中：

命令最后SWEnvRunningTasks部分可以修改为：SWEnvStartupPrograms（获取启动程序相关信息）、SWEnvEnvVars（获取环境变量）

taskkill /im(pid) imagename(pid) /f：Windows自带进程终止程序,/f强制终止程序，imagename(pid)可以用tasklist获取

pskill pid(进程名字)：为pstool的一个子程序，与taskkill一样，第一次运行需要运行pskill /accepteula命令

net user username password /add：添加用户

net localgroup Administrators username /add：将用户添加到管理员组，添加用户和添加到组可以写在一起：

net localgroup adminstrators：查看管理员组用户

net user username ：查看指定用户的信息

net user username /active:yes：激活用户

net user username password：修改用户密码
